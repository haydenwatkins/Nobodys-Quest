🎨 Sprite Lab

The game draws characters from little letter grids right in the code — not picture files. Piskel only understands pictures. This page translates between the two, so you can pull an existing sprite into Piskel, redraw it, and bring it back.

Step 1 — Export a sprite as a picture

Pick a character, then download its frame(s) as PNGs.

⚠️ These pictures are TINY on purpose (the same size the game uses). That's normal! In Piskel, use the zoom/magnifying glass to work up close.

Step 2 — Edit it in Piskel

  1. Go to piskelapp.com → Create Sprite
  2. Menu → Import → choose the PNG you downloaded above
  3. Draw with the plain Pencil tool at 100% opacity (not the soft brush) — that keeps colors exact, which matters for Step 3
  4. If your sprite has 2 frames, edit each PNG as its own separate Piskel file
  5. When you're happy: menu → Export → PNG → download it
